O que é o Protocolo MCP?

O Model Context Protocol (MCP) permite que seus agentes acessem ferramentas complexas (como busca em bancos de dados, execução de código ou acesso a APIs locais) de forma segura. Selecione um agente abaixo para acoplar ou remover servidores.
