Como funciona

O scheduler (APScheduler) roda no mesmo processo do FastAPI e dispara tarefas no horário definido. Tarefas pendentes são repopuladas no startup a partir do banco — sobrevivem a reinícios.

Tipos de ação

  • enviar_mensagem: texto direto para o telefone
  • rodar_ferramenta: executa ferramenta cadastrada
  • callback_agente: re-injeta prompt no agente como se fosse do usuário
