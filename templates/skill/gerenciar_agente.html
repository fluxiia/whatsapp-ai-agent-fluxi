Como funciona

As skills selecionadas são listadas no system prompt do agente (só nome + descrição). Quando o LLM identifica que uma skill é relevante, ele a invoca via invocar_skill e recebe as instruções completas.

Dica

Evite adicionar muitas skills com descrições similares — o LLM pode ter dificuldade de escolher. Prefira descrições únicas e específicas.
